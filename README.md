# Student Management System using Java JDBC

## 📌 Project Description

A simple Student Management System built using Java, JDBC, and MySQL to perform CRUD operations.

## 🚀 Features

* Add Student
* View Students
* Update Student
* Delete Student

## 🛠️ Technologies Used

* Java
* JDBC
* MySQL

## 💻 Tools Used

* VS Code
* MySQL Command Line Client

## 📂 Database

Database Name: studentdb

Table Name: students

## 🔗 Concepts Learned

* JDBC Connectivity
* SQL Queries
* PreparedStatement
* CRUD Operations
* Exception Handling
